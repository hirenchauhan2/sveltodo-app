# Sveltodo App - Todo list app

Learning Svelte by building projects.

## Made with
- Svelte + Vite template by `yarn create vite my-svelte-app --template svelte`
- TailwindCSS for styling

## Get started

### Install deps and start dev server using yarn

```
yarn && yarn dev
```
